# SHUSH - Ultimate Stealth

**SHUSH** — это динамичный 2D стелс-экшен, написанный на Python с использованием Pygame. Вы играете за профессионального вора, чья задача — проникать на хорошо охраняемые объекты, выносить ценные произведения искусства и сбегать незамеченным. 

Действуйте тихо, прячьтесь в тени, следите за выносливостью и помните: один неверный шаг, и за вами выедет спецназ!

## Особенности игры
* **Продвинутый стелс:** Охранники реагируют на звук ваших шагов (особенно при беге) и имеют динамические конусы зрения, которые блокируются стенами.
* **Система паники:** Если вас заметили, у вас есть лишь пара десятков секунд, чтобы добежать до зоны эвакуации, прежде чем на уровень ворвется непробиваемый отряд SWAT.
* **Встроенный Редактор Уровней:** Создавайте собственные сценарии для ограблений! Рисуйте стены, расставляйте камеры, лут и патрули, а затем запускайте их прямо из главного меню через кнопку «Собственный уровень».
* **Менеджмент ресурсов:** Контролируйте свою шкалу выносливости. Бежать вечно не выйдет!

## Запуск игры
Для запуска вам понадобится установленный Python 3 и библиотека Pygame.

1. Склонируйте репозиторий:
   ```bash
   git clone [https://github.com/Denratnegeiten/SHUSH.git](https://github.com/Denratnegeiten/SHUSH.git)
   ```
2. Перейдите в папку с игрой:
   ```bash
   cd SHUSH
   ```
3. Установите зависимости:
   ```bash
   pip install pygame
   ```
4. Запустите игру:
   ```bash
   python main.py
   ```
